# 議事録 — 2026-04-15

## 概要
GitHub Issue #1 をもとに **Personal Daily News v1** を設計・実装した。
4つのニュースソースから記事を収集し、Claude Code スケジュールタスクで要約・Kazuki 向けにエンリッチして Gmail に HTML メール配信するシステム。

---

## 1. 要件定義

Issue #1 から読み取った要件：

| ソース | 方式 | 件数 |
|---|---|---|
| Hacker News | Firebase API | Top 10 |
| Reddit | JSON API | 各5件（LocalLLaMA / nextjs / machinelearning） |
| ITmedia | RSS (feedparser) | 最新10件 |
| Product Hunt | RSS | 最新5件 |

- Claude API で要約 → Gmail（smtplib）に毎朝配信

---

## 2. 設計上の決定事項

| 項目 | 決定 | 理由 |
|---|---|---|
| **要約エンジン** | Claude Code スケジュールタスク | Anthropic API 別途課金を避け Max プラン枠内で運用 |
| **メール形式** | HTML | 可読性・カテゴリ分類の視認性 |
| **要約粒度** | per-source highlight → enriched（後述） | 当初はソース単位、後にパーソナライズ化 |
| **並列取得** | `ThreadPoolExecutor` | 4ソースの待ち時間短縮 |
| **ITmedia RSS URL** | `news_bursts.xml` に修正 | 当初の `itmakoto.xml` が 404 にリダイレクト |

## 8. トラブルシューティング記録

| 問題 | 原因 | 解決策 |
|---|---|---|
| `pip` コマンドが見つからない | macOS の Homebrew 管理 Python | `pip3` または `python3 -m pip` を使用 |
| `pip3 install` が externally-managed エラー | macOS PEP 668 制約 | `python3 -m venv .venv` で仮想環境を作成 |
| ITmedia 0件 | RSS URL (`itmakoto.xml`) が 404 | `news_bursts.xml` に修正 |
| `ANTHROPIC_API_KEY` 不要に | Claude Code Cron に切り替え | `.env` から削除可 |
